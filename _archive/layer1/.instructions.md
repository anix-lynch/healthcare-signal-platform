## 🚀 Common Commands

```bash
# 1. Start REST API (55K patient records)
./scripts/start_api.sh
# → http://localhost:8000/docs

# 2. Compile dbt warehouse
cd dbt-project
dbt compile --select fact_patient_encounters
dbt run --profiles-dir .

# 3. View TMDL semantic model (no GUI)
cat powerbi-model/tables/Patient\ Encounters.tmdl

# 4. Train ML model
cd ml-pipeline/src
python train.py

# 5. Verify all resume claims
./scripts/resume-proof-verification.sh

# 6. Diagnostic check (Fabric/Azure auth)
./scripts/fabric_doctor.sh
```
